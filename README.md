# tlake8

[flake8](https://github.com/PyCQA/flake8) for tab junkies

## Getting Started

1. Clone it:

```sh
$ git clone git://github.com/hyperair/tlake8
```

2. Install it via pip:

```sh
$ pip install -e .
```

3. Run it:

```sh
$ tlake8 --help
```

### Prerequisites

- Depends on flake8, which it is monkey-patching in the  first place.
